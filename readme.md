# Juego Piedra, Papel o Tijera

Este es un pequeño proyecto hecho en Python donde se puede jugar Piedra, Papel o Tijera contra la computadora.

## Cómo funciona
El usuario escribe su opción y la computadora elige una al azar. Luego el programa compara las dos opciones y muestra quién gana.

## Herramientas usadas
- Spyder

## Lógica del programa
Se usan condiciones if, elif y else para comparar las jugadas.
